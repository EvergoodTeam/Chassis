## Adding to your own project

```
repositories {

	maven {
		url "https://jitpack.io"
		content { includeGroup "com.github.evergoodteam" }
	}
}
```

```
dependencies {

	//Chassis
	modImplementation 'com.github.evergoodteam:chassis:<releaseVersion>'
}
```
